# Smart Recipe Code Samples

This directory contains C# backend code samples related to smart recipe generation and recipe management.

## Included Files

- `SmartRecipesController.cs`: Smart recipe API entry point, including daily recipe retrieval, forced regeneration, and Server-Sent Events streaming for recipe generation.
- `RecipeService.cs`: Recipe business logic, including recipe creation, listing by scope, detail access control, tag handling, ingredient persistence, updates, and deletion.

These files demonstrate authenticated smart recipe workflows, household-aware recipe access, AI-generated recipe delivery, streaming responses, transactional recipe creation, tag normalization, and recipe visibility rules.
